### 1 - Apresentação

**Apresentação**

- Alura Newsletter

- Documentação
    - Link: https://pt-br.legacy.reactjs.org/
    - Node: https://nodejs.org/en
    - Vite: https://vitejs.dev/

- Yarn (Globalmente)
```
npm install --global yarn
```    

- Criar o projeto react com vite
```
yarn create vite 
```

- Nome do projeto: newsletter_01
    - Tecnologia: React
    - Com: JavaScript


- build (node_modules)
```
yarn
```

- Roda o projeto
```
yarn dev
```

- Localhost:
    - Link: http://localhost:5173/
